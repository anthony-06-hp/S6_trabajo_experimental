# Guía Práctica 2 - Algoritmos de Búsqueda y Optimización

En esta práctica se estudian e implementan diferentes algoritmos utilizados en Inteligencia Artificial para la búsqueda de soluciones y optimización de problemas.

Se analizaron conceptos relacionados con grafos, rutas, búsquedas informadas y no informadas, además del funcionamiento de algoritmos genéticos.
